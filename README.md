# 🚀 B2B SaaS Business Intelligence (KBT Project)

Platform intelijen bisnis *All-in-One* yang dirancang khusus untuk membantu UMKM dan penjual *online* beralih menjadi bisnis yang digerakkan oleh data (*data-driven*). 

Proyek ini dikembangkan untuk memenuhi tugas mata kuliah Kewirausahaan Berbasis Teknologi (KBT).

## 🌟 Modul Utama

1. **Dashboard Penjualan:** Pusat pencatatan riwayat transaksi dan visualisasi metrik finansial secara *real-time*.
2. **Optimasi Marketplace:** Alat inteligensi kompetitif untuk meningkatkan *Search Engine Optimization* (SEO) produk melalui *scoring* *listing* dan analisis *keyword*.
3. **Konsultan Bisnis Berbasis Data:** Fitur analitik tingkat lanjut yang mencakup *Sales Forecasting* (pemodelan *time-series*) dan *A/B Testing* secara statistik.

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript
- **Styling:** Tailwind CSS
- **Visualisasi Data:** Recharts
- **Deployment:** Vercel
